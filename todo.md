# 目的

ブラウザで動作するTetrisを作りたいです

# 要件

- ブラウザ上で動作する
- JavaScriptで実装する
- ブロックの回転、移動、落下ができる
- ラインが揃ったら消える
- スコアが表示される
- ゲームオーバーの判定がある
- シンプルなUIで操作しやすい
- キーボード操作に対応する（矢印キーで移動・回転、スペースキーで一気に落下など）
- レスポンシブデザインで、PCとモバイルの両方で快適にプレイできる
- ゲームの開始、停止、リセットができるボタンを用意
- サウンドエフェクトやBGMを追加して、ゲーム体験を向上させる
- ゲームの難易度設定ができる（速度の調整など）
- ハイスコアの保存機能を実装する（ローカルストレージを使用）
- チュートリアルやヘルプセクションを用意して、初心者でも遊びやすくする

# 手順

- Gitを使ってバージョン管理を行う
- GitHub Pagesを使って公開する
- HTML、CSS、JavaScriptで実装する
- 必要に応じてライブラリやフレームワークを使用する
- こまめにコミットする
