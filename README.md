# Tetris Game 🎮

ブラウザで動作するクラシックなTetrisゲームです。

## 🎯 特徴

- ✨ シンプルで直感的なUI
- 📱 レスポンシブデザイン（PC・モバイル対応）
- 🎮 キーボード操作とタッチ操作の両方に対応
- 🔊 サウンドエフェクト付き
- 💾 ハイスコア保存機能（ローカルストレージ）
- 🎚️ 難易度設定（簡単・普通・難しい）
- 📊 スコア・レベル・ライン数の表示

## 🕹️ 操作方法

### キーボード操作
- `←` / `→` : ブロックの左右移動
- `↑` : ブロックの回転
- `↓` : 高速落下
- `Space` : 即座に落下

### モバイル操作
画面下部のボタンで操作できます。

## 🚀 遊び方

1. 「スタート」ボタンをクリックしてゲーム開始
2. 落下してくるブロックを操作して横一列を揃える
3. 列が揃うと消えてスコアが加算される
4. ブロックが上まで積み重なるとゲームオーバー

## 🎨 実装技術

- HTML5
- CSS3
- Vanilla JavaScript
- Canvas API
- Web Audio API（効果音）
- Local Storage（ハイスコア保存）
